HU-05 · Avance por etapas · ligar programa por versión

Programa de obra por etapa (versión del programa)

Mockup (no es el sistema). Datos reales de SOP-2026-001: v1 original (Σ 25 600, $2 400 000) → Convenio CM-001 (AD-01 balizamiento +5 000 m²) → v2 vigente (Σ 30 600, $2 600 000).

Propuesta B ya elegida (dos etapas: original congelado + vigente). Novedad de este mockup: desde cada etapa se puede «Ver programa de obra» ligado a SU versión — v1 muestra el plan original (sus fechas, conceptos, cantidades); v2 el plan modificado (con el adicional). Así se compara "así era el plan original" vs "así quedó con el convenio". Abajo van 2 variantes del desplegable — elige una.
Nota honesta de datos: este convenio fue de monto (no de plazo), por eso v2 conserva las mismas fechas de v1 y solo añade el concepto AD-01. La infraestructura guarda fechas por versión (programa_version_celda), así que un convenio de plazo mostraría fechas distintas en v2 automáticamente.
1
