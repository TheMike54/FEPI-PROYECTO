# IA

Indice de documentos dirigidos a IA o utiles para que una instancia futura retome el proyecto.

Este acomodo aplica la **Propuesta 2**: los archivos no se mueven de sus ciclos; aqui solo se concentran enlaces, orden de lectura y clasificacion.

## Orden recomendado

1. Leer `IA_DIRECTA.md`.
2. Abrir primero los documentos marcados como contexto/retoma.
3. Usar `IA_MIXTA.md` solo cuando haga falta ampliar contexto.

## Indices

- `IA_DIRECTA.md`: documentos con prompts, handoff, sesiones autonomas o instrucciones explicitas para IA.
- `IA_MIXTA.md`: documentos principalmente humanos, pero utiles para IA por contexto, reglas o continuidad.
